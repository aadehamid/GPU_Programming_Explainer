# Code samples for EuroCC workshop on GPU programming with CUDA[^1]

This repository contains code samples, which we will use during the workshop on GPU programming with CUDA.

## Installation
Download the code using:

```console
$ git clone https://repo.sling.si/dsluga/programming-cuda.git
```

and move to subdirectory:

```console
$ cd programming-cuda
```
